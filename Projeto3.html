Informações do Projeto - Sistema de Vendas de Veículos

  • Categoria: Desenvolvimento SQL / Sistema Comercial
  • Cliente: Projeto Acadêmico
  • Data do Projeto: 2025
  • Descrição: Modelo de banco de dados que controla pedidos e vendas de veículos. As entidades principais são Vendedor, Cliente, Veículo, Pedido e Pessoa. Cada pessoa pode ser cliente ou vendedor. O Pedido relaciona um veículo vendido a um cliente e o vendedor responsável, armazenando valores de venda e comissão. O sistema permite rastrear todas as transações realizadas.

Informações do Projeto - Banco de Dados Academia

  • Categoria: Modelagem de Dados / MySQL
  • Cliente: Projeto Acadêmico
  • Data do Projeto: 2025
  • Descrição: Projeto de modelagem de um banco de dados para uma academia. O diagrama define entidades como Aluno, Treino, Plano, Pagamento, Instrutor e Pessoa, além de tabelas associativas para representar os relacionamentos entre alunos e treinos. O objetivo é estruturar um sistema que registre matrículas, planos, pagamentos, especialidades dos instrutores e os treinos atribuídos a cada aluno.

Informações do Projeto - Sistema de Vacinação

  • Categoria: Desenvolvimento SQL / Modelagem de Dados
  • Cliente: Projeto Acadêmico
  • Data do Projeto: 2025
  • Descrição: Modelo de banco de dados que gerencia a aplicação de vacinas. O sistema possui tabelas para Paciente, Profissional, Pessoa_Física, Vacina e Aplicacao_Vacina. Cada aplicação relaciona um paciente, um profissional e uma vacina, registrando a data da aplicação. A tabela Pessoa_Física centraliza informações pessoais compartilhadas entre pacientes e profissionais.
